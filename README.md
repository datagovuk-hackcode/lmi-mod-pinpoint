### Data used


* LMI for All API

---

 
### How to run
 
Make sure you have [Xcode], [git], and [CocoaPods] installed
 
```sh
git clone git@github.com:rewiredstatedevs/lmi-mod-pinpoint.git
cd lmi-mod-pinpoint
pod install
```
 
and then run Pinpoint.xcworkspace in Xcode.
