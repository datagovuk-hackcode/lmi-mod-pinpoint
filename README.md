### How to run
 
Make sure you have [Xcode], [git], and [CocoaPods] installed
 
```sh
git clone git@github.com:rewiredstatedevs/LMI-for-all-mod-pinpoint2.git
cd LMI-for-all-mod-pinpoint2
pod install
```
 
and then run Pinpoint.xcworkspace in Xcode.
